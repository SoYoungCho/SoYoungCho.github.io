Multilingual Speech Synthesis – Samples

See Github of this work for further details and source code or visit interactive demo notebooks for code switching, voice cloning and multilingual training.

We compared the abilities of three multilingual text-to-speech models based on Tacotron 2.

  • The first (called shared) shares the whole encoder and uses an adversarial classifier to remove language-dependent information. The idea comes from Learning to speak fluently in a foreign language: Multilingual speech synthesis and cross-language voice cloning (here).
  • The second (called separate) has separate encoders for each language. It closely follows End-to-End Code-Switched TTS with Mix of Monolingual Recordings (here).
  • Finally, the third (called generated) combines the best of both approaches, i.e., effective parameter sharing of the first method and flexibility of the second. It has an encoder with language-specific parameters generated by a parameter generator network and it also makes use of the adversarial process.

We use the WaveRNN model for vocoding.
